# EMR-
EMR (Electronic Medical Record) adalah versi digital dari rekam medis berbasis kertas di klinik atau fasilitas kesehatan. Sistem ini memuat riwayat medis, diagnosis, dan perawatan pasien secara spesifik di satu lokasi, dirancang untuk menggantikan pencatatan manual agar akses data lebih cepat, akurat, dan aman.
